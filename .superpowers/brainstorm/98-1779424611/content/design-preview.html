设计方案预览

底层强调色：荧光绿 → 荧光红

改动范围

仅修改 style.css 中 .layer-bottom 的一个 CSS 变量

style.css — .layer-bottom 修改
.layer-bottom {
  ...
  --neon: #cbf902 #ff2d2d;  /* 荧光绿 → 荧光红 */
  --neon-dim: rgba(203, 249, 2, 0.12) rgba(255, 45, 45, 0.12);
  --neon-glow: rgba(203, 249, 2, 0.3) rgba(255, 45, 45, 0.3);
  --neon-ghost: rgba(203, 249, 2, 0.05) rgba(255, 45, 45, 0.05);
  --border-active: rgba(203, 249, 2, 0.25) rgba(255, 45, 45, 0.25);
}
